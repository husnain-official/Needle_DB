-------pipeline/embedder.py-------
Requirements: 
    1. 'ollama' installed
    2. Use 'ollama' to pull 'nomic-embed-text'
    3. Have 'dotenv' installed to use .env files
        for this you have to create a virtual env,
            python3 -m venv .venv       # Create the virtual env
            source .venv/bin/activate   # activate the venv
            pip install dotenv          # Install the package
            deactivate                  # To exit the venv
